Frequently Asked Questions

Is my file uploaded to a server?

No. Your file is never uploaded anywhere. All password protection and encryption happen locally in your browser using the Web Crypto API. Your data never leaves your device.

What encryption is used to protect my file?

UniDoc uses AES-256-GCM encryption with PBKDF2-SHA256 key derivation (150,000+ iterations). This is the same military-grade standard used by governments and financial institutions.

Can I unlock or decrypt my file later?

Yes. You can decrypt your password-protected .ufenc file anytime using the UniDoc Encrypt tool or the UniDoc mobile app. Just provide the same password you used during encryption.

Is this password protection tool free?

Yes, completely free with no registration required. For advanced features like vault storage, biometric lock, and QR code key sharing, download the UniDoc app.

Does the file protection tool work offline?

Yes. Once the page is loaded, all encryption operations run entirely in your browser using JavaScript. You can disconnect from the internet and the tool will continue to work.

What file types can I password protect?

Any file type — PDF, Word, Excel, images, videos, archives, and more. The tool works with any file up to 1 GB.

Is the protected file compatible with the UniDoc app?

Yes. Files encrypted here use the exact same .ufenc format as the UniDoc mobile app. You can encrypt on the web and decrypt on your phone, or vice versa.
